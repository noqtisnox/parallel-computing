# Паралельні обчислення і розподілені системи: Лабораторні роботи

Репозиторій містить цикл лабораторних робіт з дослідження методів паралельного програмування.

## Лабораторна робота №1
**Тема:** Способи розпаралелювання та організації обчислень. Послідовні алгоритми.

**Завдання:** Реалізувати розв’язання системи лінійних алгебраїчних рівнянь методом Гаусса (прямий та зворотний хід) у послідовному та паралельному режимах.

### Реалізація
- **Мова:** C++
- **Бібліотеки:** `std::thread` для паралелізації, `chrono` для вимірювання часу.
- **Особливості:**
  - Розпаралелювання прямого ходу (forward elimination) за рядками.
  - Динамічний розподіл навантаження між потоками залежно від кількості ядер CPU.
  - Генерація вхідних даних за допомогою Python-скрипта.

### Вимоги до оточення
- **OS:** Linux (через WSL)
- **Compiler:** g++ з підтримкою C++20
- **Flags:** `-pthread` (обов'язково для роботи з потоками на Linux)

### Як запустити
1. Згенеруйте дані: `python3 scripts/generate_input_data.py`
2. Скомпілюйте проєкт:
   ```bash
   g++ -O3 main.cpp ./src/*.cpp -o gauss_solver -pthread
   ```
3. Запустіть бінарник:
    ```bash
    ./gauss_solver
    ```
